# Python Stack - Basic to Advanced

This repository contains a single Python script demonstrating stack data structure concepts from basic usage to more advanced examples.

## Contents

- `PYTHON STACK - BASIC TO ADVANCED.py` - a sample script showing:
  - basic stack operations using a Python list
  - push, pop, peek, size, and empty-check operations
  - an object-oriented stack implementation using a `Stack` class
  - stack usage with `collections.deque`
  - real-world stack examples such as browser history and undo functionality
  - a decimal-to-binary conversion implemented with a stack

## Purpose

This project is intended for learners who want a simple, hands-on introduction to stack behavior in Python. It demonstrates the LIFO (Last In, First Out) pattern and how stacks can be used in different situations.

## How to run

1. Ensure you have Python installed.
2. Open a terminal in this folder.
3. Run:

```bash
python "PYTHON STACK - BASIC TO ADVANCED.py"
```

## What you will learn

- How to implement stack operations with built-in Python list methods
- How to define a stack class with methods for common stack behaviors
- How `deque` can also represent a stack
- Practical examples of stacks in real-world-like tasks
- How to use a stack to convert a decimal number to binary

## Notes

- The primary demonstration file is self-contained and does not require external libraries beyond Python standard library.
- This project is best used as a learning exercise or reference for basic stack patterns in Python.
